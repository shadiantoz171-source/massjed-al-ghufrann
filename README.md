# نظام تسميعات مسجد الغفران 🌙

تطبيق ويب متقدم لإدارة تسميعات الطلاب من القرآن الكريم في المسجد.

## المميزات الرئيسية

### الأدوار:
- **المدير**: إدارة كاملة للمعلمين والطلاب والتسميعات
- **المعلم**: إنشاء وإدارة بيانات الطلاب والتسميعات
- **الطالب**: عرض بياناته الشخصية والتسميعات فقط

### نظام التسميعات:
- الأجزاء 1-29: تسميع صفحة تلو صفحة
- الجزء 30: تسميع حسب السور
- نظام تلوين ذكي:
  - 🟨 أصفر: تسميعات قديمة
  - 🟩 أخضر: تسميعات حديثة
- إعادة تعيين سنوي تلقائي

## التقنيات المستخدمة

### Backend:
- Node.js + Express
- PostgreSQL
- JWT Authentication
- bcrypt for password hashing

### Frontend:
- React
- Tailwind CSS
- React Router
- Axios

## التثبيت والتشغيل

```bash
# تثبيت المتطلبات
npm install

# تعيين متغيرات البيئة
cp .env.example .env

# تشغيل قاعدة البيانات
npm run db:migrate

# تشغيل الخادم
npm start
```
