# واجهة الويب (Frontend)

قمت بإنشاء واجهة MVP بسيطة باستخدام Vue 3 وVite وTailwindCSS (RTL) تحت المجلد `frontend/` في المستودع.

محتويات أساسية:
- صفحات: تسجيل دخول (محاكاة)، لوحة المدير، لوحة المعلم، صفحة عرض طالب.
- إعداد Tailwind وملف CSS أساسي يدعم RTL من خلال `dir="rtl"` في index.html.
- التوجيه الأساسي Vue Router لصفحات التطبيق.

تشغيل محليًا:
1. انتقل إلى مجلد `frontend`:
   cd frontend
2. ثبت الحزم:
   npm install
3. شغّل البيئة التطويرية:
   npm run dev
4. افتح المتصفح على http://localhost:5173

ملاحظات وقيود حالية:
- المصادقة والاتصال بالـAPI لم تُفعّل بعد؛ تسجيل الدخول حاليًا يعيد توجيهاً محاكياً حسب الدور.
- يجب تهيئة واجهات API في backend (Laravel) ثم ربط الواجهات بـaxios لاستدعاء البيانات الحقيقية.

ما أفعله لاحقًا عند طلبك:
- ربط الواجهات مع API (إضافة axios instances وendpoints)، وحماية الصفحات عبر جلسات أو tokens.
- إنشاء قوائم CRUD فعلية للطلاب والمعلمين والتسميعات.
